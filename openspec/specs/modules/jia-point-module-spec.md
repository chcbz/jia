# Point 积分模块规格文档

## 模块概述

Point 积分模块提供用户积分管理和交易记录功能。

- **子模块**: api, core, mapper, service
- **包名**: cn.jia.point
- **版本**: 1.1.2-SNAPSHOT

## 核心功能

### 1. 积分账户

| 功能 | 说明 |
|------|------|
| 账户创建 | 用户积分账户 |
| 账户查询 | 查询用户积分 |
| 积分余额 | 实时余额 |

### 2. 积分获取

| 功能 | 说明 |
|------|------|
| 任务奖励 | 完成任务的奖励 |
| 活动奖励 | 参与活动的奖励 |
| 充值 | 积分充值 |
| 邀请奖励 | 邀请新用户奖励 |

### 3. 积分消费

| 功能 | 说明 |
|------|------|
| 兑换 | 商品/服务兑换 |
| 抵扣 | 订单抵扣 |
| 退款 | 退款返还积分 |

### 4. 积分记录

| 功能 | 说明 |
|------|------|
| 收入记录 | 积分获取记录 |
| 支出记录 | 积分消费记录 |
| 明细查询 | 积分变动明细 |

## 相关模块

- **jia-common-core**: 提供基础实体
- **jia-user-core**: 用户信息
