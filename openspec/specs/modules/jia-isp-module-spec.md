# ISP ISP服务模块规格文档

## 模块概述

ISP ISP服务模块提供ISP相关的服务管理功能。

- **子模块**: api, core, mapper, service, starter
- **包名**: cn.jia.isp
- **版本**: 1.1.2-SNAPSHOT

## 核心功能

### 1. 品牌管理

| 功能 | 说明 |
|------|------|
| 品牌列表 | 汽车品牌管理 |
| 品牌版本 | 品牌车型版本 |
| 品牌分类 | 品牌分类管理 |

### 2. 车辆管理

| 功能 | 说明 |
|------|------|
| 车辆信息 | 车辆基本信息 |
| 车辆查询 | 车辆查询接口 |
| 车辆绑定 | 用户车辆绑定 |

### 3. 服务接口

| 功能 | 说明 |
|------|------|
| 车辆查询API | 第三方车辆查询 |
| 数据同步 | 数据同步接口 |
| 服务集成 | 外部服务集成 |

## 相关模块

- **jia-common-core**: 提供基础实体
- **jia-user-core**: 用户信息
