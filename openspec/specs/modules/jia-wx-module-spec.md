# WX 微信模块规格文档

## 模块概述

WX 微信模块提供微信公众号和小程序集成功能。

- **子模块**: api, core, mapper, service
- **包名**: cn.jia.wx
- **版本**: 1.1.2-SNAPSHOT

## 核心功能

### 1. 公众号功能

| 功能 | 说明 |
|------|------|
| 消息接收 | 接收用户消息 |
| 消息回复 | 自动回复消息 |
| 菜单管理 | 自定义菜单 |
| 用户管理 | 关注用户管理 |
| 图文消息 | 图文消息发送 |

### 2. 微信支付

| 功能 | 说明 |
|------|------|
| 支付下单 | 发起支付 |
| 支付通知 | 支付结果通知 |
| 退款 | 申请退款 |
| 对账 | 交易对账 |

### 3. 微信授权

| 功能 | 说明 |
|------|------|
| 网页授权 | OAuth2.0授权 |
| 获取OpenID | 获取用户OpenID |
| 获取用户信息 | 获取用户资料 |

### 4. 小程序

| 功能 | 说明 |
|------|------|
| 登录 | 小程序登录 |
| 授权 | 用户授权 |
| 消息 | 小程序消息 |

### 5. 模板消息

| 功能 | 说明 |
|------|------|
| 模板管理 | 消息模板管理 |
| 发送 | 模板消息发送 |

## 相关模块

- **jia-common-core**: 提供基础实体
- **jia-user-core**: 用户OpenID
- **jia-material-core**: 素材管理
- **jia-sms-core**: 通知功能
