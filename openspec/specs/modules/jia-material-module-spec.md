# Material 素材管理模块规格文档

## 模块概述

Material 素材管理模块提供媒体素材、新闻、常用短语、投票等素材管理功能。

- **子模块**: api, core, mapper, service, starter
- **包名**: cn.jia.material
- **版本**: 1.1.2-SNAPSHOT

## 核心功能

### 1. 媒体素材

| 功能 | 说明 |
|------|------|
| 图片管理 | 图片上传、存储、CDN |
| 视频管理 | 视频素材管理 |
| 音频管理 | 音频素材管理 |
| 文件管理 | 通用文件存储 |

### 2. 新闻素材

| 功能 | 说明 |
|------|------|
| 文章管理 | 新闻文章CRUD |
| 分类管理 | 文章分类 |
| 标签管理 | 文章标签 |
| 审核 | 文章审核流程 |

### 3. 常用短语

| 功能 | 说明 |
|------|------|
| 短语分类 | 短语分类管理 |
| 短语维护 | 常用短语维护 |
| 快捷回复 | 客服快捷回复 |

### 4. 投票管理

| 功能 | 说明 |
|------|------|
| 投票创建 | 创建投票活动 |
| 选项管理 | 投票选项管理 |
| 投票参与 | 用户投票 |
| 统计 | 投票结果统计 |

### 5. PV统计

| 功能 | 说明 |
|------|------|
| 页面统计 | 页面访问统计 |
| 点击统计 | 元素点击统计 |
| 报表 | 访问数据分析 |

## 相关模块

- **jia-common-core**: 提供基础实体
- **jia-user-core**: 用户信息
- **jia-base-core**: 字典功能
